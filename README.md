# 🔍 탐색 알고리즘 시각화

세 가지 주요 탐색 알고리즘(순차탐색, 이진탐색, 보간탐색)을 실시간으로 시각화하여 비교할 수 있는 웹 애플리케이션입니다.

![Search Algorithm Visualizer](assets/screenshot.png)

## ✨ 주요 기능

- **세 가지 알고리즘 지원**
  - 순차탐색 (Linear Search) - O(n)
  - 이진탐색 (Binary Search) - O(log n)
  - 보간탐색 (Interpolation Search) - O(log log n)

- **실시간 시각화**
  - 현재 비교 중인 요소 강조
  - 탐색 범위 표시
  - 결과 애니메이션

- **통계 정보**
  - 단계 수 추적
  - 비교 횟수 계산
  - 탐색 결과 표시

- **인터랙티브 UI**
  - 알고리즘 전환
  - 목표값 설정
  - 탐색 시작/초기화
